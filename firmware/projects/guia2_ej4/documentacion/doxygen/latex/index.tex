\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{General Description}{General Description}}\label{index_genDesc}
Este programa implementa una aplicación de adquisición y transmisión de señales analógicas utilizando un microcontrolador con Free\+RTOS.


\begin{DoxyItemize}
\item Se toma una entrada analógica mediante el canal CH1.
\item La señal adquirida se envía a la PC por UART (puerto serie) para su visualización.
\item En paralelo, una segunda tarea genera una señal ECG simulada y la envía por una salida D/A.
\item Ambas tareas se sincronizan mediante temporizadores independientes (TIMER\+\_\+A y TIMER\+\_\+B).
\end{DoxyItemize}
