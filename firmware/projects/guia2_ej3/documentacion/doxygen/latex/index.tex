\chapter{Sensor de distancia ultrasónico con Free\+RTOS }
\hypertarget{index}{}\label{index}\index{Sensor de distancia ultrasónico con FreeRTOS@{Sensor de distancia ultrasónico con FreeRTOS}}
\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{Descripción general}{Descripción general}}\label{index_genDesc}
Este programa utiliza Free\+RTOS para gestionar tareas relacionadas con la medición de distancia mediante un sensor ultrasónico {\bfseries{HC-\/\+SR04}}, junto con el control de {\bfseries{LEDs indicadores}}, una {\bfseries{pantalla LCD}} y comunicación {\bfseries{UART}}.

El sistema emplea temporizadores e interrupciones para sincronizar las tareas, mientras que los {\bfseries{switches}} y la {\bfseries{UART}} permiten modificar el modo de funcionamiento\+:
\begin{DoxyItemize}
\item {\bfseries{Encendido/\+Apagado}} del sistema.
\item {\bfseries{Hold}} para mantener el valor actual de distancia en pantalla.
\end{DoxyItemize}
